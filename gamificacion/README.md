# 🎮 Sistema de juego del curso

Porque un curso de DevOps para profes debe ser **la clase más divertida del año**.
Cada laboratorio es una **misión**. Completarla da **XP** y desbloquea una **medalla**.

## 🏅 Rangos

| Rango | XP necesaria |
| ----- | ------------ |
| 🥚 Aprendiz | 0 |
| 🐳 Operador de Contenedores | 150 |
| ⚙️ Ingeniero de Entrega | 350 |
| ☸️ Pilot@ de Kubernetes | 600 |
| 🏛️ Arquitect@ DevOps | 900 |

## ⭐ Reglas

1. **XP base** por entregar la evidencia de la misión (captura + comando).
2. **+ Bonus** por los retos extra de cada laboratorio.
3. La **medalla** se desbloquea al completar la misión; se marca en la
   [tarjeta de logros](tarjeta-logros.md) imprimible.
4. La capa que hace que esto valga como **capacitación docente**: en cada misión
   respondes *"¿Cómo lo convierto en práctica para mis alumnos?"* → eso vale **+15 XP**.

## 🎲 Dinámicas opcionales para el instructor

- **Tablero proyectado:** lleva el conteo de XP por equipo en el pizarrón o una hoja.
- **Insignia física:** imprime stickers de las medallas y pégalos en la tarjeta.
- **Sello de "primer /health verde":** quien levante primero la app gana +10 XP sorpresa.
- **Modo cooperativo:** los equipos que terminan ayudan a otros → +10 XP por rescate.
