# 🎮 The course's game system

Because a DevOps course for teachers should be **the most fun class of the year**.
Every lab is a **mission**. Completing it gives **XP** and unlocks a **medal**.

## 🏅 Ranks

| Rank | XP needed |
| ----- | ------------ |
| 🥚 Apprentice | 0 |
| 🐳 Container Operator | 150 |
| ⚙️ Delivery Engineer | 350 |
| ☸️ Kubernetes Pilot | 600 |
| 🏛️ DevOps Architect | 900 |

**Total available: 1,280 XP.** Details for each mission in [`misiones.md`](misiones.en.md).

## ⭐ Rules

1. **Base XP** for submitting the mission's evidence (screenshot + command).
2. **+ Bonus** for each lab's extra challenges.
3. The **medal** is unlocked when you complete the mission; it's marked on the
   printable [achievement card](tarjeta-logros.en.md).
4. The layer that makes this count as **teacher training**: in every mission
   you answer *"How do I turn this into practice for my students?"* → that's worth **+15 XP**.

## 🎲 Optional dynamics for the instructor

- **Projected board:** keep the XP count per team on the whiteboard or a sheet.
- **Physical badge:** print medal stickers and stick them on the card.
- **"First green /health" stamp:** whoever brings the app up first wins a surprise +10 XP.
- **Co-op mode:** teams that finish help others → +10 XP per rescue.
