## Regenerate / edit

```bash
cd slides
npm install            # first time only
node generate.js       # 8 instructor decks (4 ES + 4 EN)
node infographics.js   # 2 infographic decks (ES + EN)
```

> The design (colors, fonts, layouts) lives in `lib.js`. The bilingual content uses
> `tr("es", "en")` in `generate.js` and `infographics.js`, so both languages come from a single source.
